# Ecological Data Visualization

## Project Overview
This project analyzes the Palmer Penguins dataset to explore species diversity and morphological patterns among Antarctic penguins. 

## Technical Implementation
- **Data Cleaning:** Used `pandas` to handle missing values and structure the dataset for analysis.
- **Statistical Analysis:** Generated summaries of body mass and bill dimensions across three species (Adelie, Chinstrap, Gentoo).
- **Interactive Visualization:** Developed a Plotly scatter plot to visualize the relationship between bill depth and length, using point size to represent body mass.

## How to Reproduce
1. Install dependencies: `pip install -r requirements.txt`
2. Run the Jupyter Notebook: `Ecological_Analysis.ipynb`
